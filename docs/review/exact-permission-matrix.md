## Explicitly Excluded Authorization

No row requests publishing, upload, comments/comment management, messaging, ads, business management, app events, demographics, webhooks, monetary/revenue data, profile/email identity, Google Drive/Gmail, broad Cloud, or administrative mutation. A feature needing any such access requires a separate authorization decision and provider review; it is not an extension of these rows.
