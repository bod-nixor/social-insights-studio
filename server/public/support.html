Support

Last reviewed: July 17, 2026

For help with Social Insights Studio, email b.o.d@nixorcollege.edu.pk.

What to include

  • The workspace or Looker Studio data source you are using.
  • The provider involved, such as TikTok, Instagram, Facebook Pages, YouTube, or Website Analytics.
  • The approximate time of the issue and the action you were taking.
  • A screenshot of the visible error if it does not contain private tokens or secrets.

Security requests

Do not send access tokens, client secrets, authorization codes, refresh tokens, passwords, or private provider console screenshots by email. If a request requires sensitive details, support will provide a safer collection path.

Deletion and disconnect requests

To disconnect a provider or request deletion of stored provider data, use the instructions on the data deletion page.
