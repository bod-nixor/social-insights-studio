Service status and known limitations

Last reviewed: July 18, 2026

TikTok dashboard and Looker Studio connector

Current implemented connector and dashboard source. A connected TikTok account with no posts can return valid profile metrics and no video rows.

YouTube channel analytics

The read-only integration is implemented but disabled by default until Google OAuth configuration and verification are complete. YouTube reports may have a visible data-through delay, and unavailable metrics are not treated as zero.

Facebook Page and Instagram professional analytics

The read-only integrations are implemented but disabled by default until exact Meta permissions, Facebook Login for Business configuration, provider review evidence, and eligible live test resources are confirmed. Instagram is independently blocked unless its exact four-scope Facebook Login set is available.

Website Analytics

The read-only GA4 integration is implemented but disabled by default until a dedicated Google OAuth client, Analytics Admin/Data APIs, consent configuration, an eligible test property, review evidence, and live verification are complete. Reporting delay, unavailable combinations, and privacy thresholding remain visible rather than being estimated.

Report PDFs

PDF reports are available only when private report storage and background generation are enabled. Reports use stored analytics snapshots, retain provider metric definitions, and expire seven days after generation.
